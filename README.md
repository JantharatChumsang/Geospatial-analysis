## Result

จากการวิเคราะห์ข้อมูลเชิงพื้นที่พบว่าถ้าเราสามารถหาทำเลที่เหมาะสมในการสร้างศูนย์ป้องกันอุบัติเหตุฉุกเฉินได้ เราก็จะสามารถเตรียมพร้อมในการรับมือกับอุบัติเหตุที่จะเกิดขึ้นได้
